Pengaturan Website

Integrasi Telegram (Auto-post Video Baru)

Setiap video berstatus "Published" (manual atau lewat auto-publish terjadwal) otomatis dikirim ke channel/grup Telegram Anda. Cara dapat Bot Token & Chat ID ada di README.md bagian Automasi.

Auto-Publish Terjadwal

Video berstatus Draft dengan Tanggal Publish yang sudah lewat akan otomatis di-publish setiap 5 menit (lewat Cloudflare Cron Trigger). Gunakan tombol ini untuk mengecek sekarang juga tanpa menunggu jadwal — berguna untuk testing.

Kirim Push Notification
